# GenerativeAdversarialNets
This project presents the design, implementation, and evaluation of a Generative Adversarial Network (GAN) for generating realistic medical images of the prostate. The model was developed in TensorFlow/Keras and trained on a dataset of approximately 2,800 uniformly formatted NumPy-stored images, with normalization applied before training to improve learning stability.

The implemented system follows a standard GAN architecture consisting of a generator and a discriminator trained in adversarial fashion. The generator progressively reconstructs images from a random latent vector using transposed convolution layers, ReLU activations, and a tanh output, while the discriminator uses convolutional layers, Leaky ReLU, and spectral normalization to improve robustness and stabilize training. The training pipeline also included checkpoint saving, image-grid visualization, and utility components such as brightness shifting to support data augmentation and monitoring.

Experimental results show a clear improvement in image quality as model capacity and training duration increased. Early experiments on a small dataset produced only coarse localization of the prostate region, whereas later versions trained on the full dataset generated more anatomically plausible axial cross-sections with better contrast and more distinct tissue boundaries. The final model, trained for 1,200 epochs, produced the most realistic outputs, and the intermediate evaluation at epoch 900 yielded SSIM 0.7344, PSNR 26.5171 dB, and MSE 0.0027, indicating strong structural similarity and acceptable image fidelity relative to the training data.

Overall, the project demonstrates that a properly configured GAN can learn meaningful anatomical patterns from prostate MRI data and generate realistic synthetic samples. The main limitation was computational cost, as training required more than 4.5 hours, suggesting that further work should focus on data augmentation, 3D model extension, transfer learning, and more powerful hardware for improved scalability and image quality
